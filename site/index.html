Security by design

Your GitHub password never enters the app.

Authorization happens on GitHub’s trusted page. GitHub Rock stores session tokens locally using Android Keystore-backed encryption and keeps sensitive operations explicit.

Privacy policySecurity policy
01
Official OAuth Device Flow

Approve access only on GitHub’s own authorization page.

02
Encrypted local tokens

Android Keystore-backed storage protects the connected session.

03
Verified release pipeline

Checksums and certificate fingerprints are produced before publication.

04
No silent branch bypass

Code changes are designed around review branches and pull requests.
